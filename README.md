# pypiano

Play generated sine wave sound with Python and Sounddevice module

## Control keys:

| Key | Desctiption |
| --- | --- |
| ESC | Exit |
| + | Increase volume |
| - | Decrease volume |
| PgUp | Octave + 1 |
| PgDn | Octave - 1 |
| Up | Current frequency + 1% |
| Down | Current frequency - 1% |

| F1-F12 | Piano note keys: |
| --- | --- |
| F1 | C |
| F2 | C# |
| F3 | D |
| F4 | D# |
| F5 | E |
| F6 | F |
| F7 | F# |
| F8 | G |
| F9 | G# |
| F10 | A |
| F11 | A# |
| F12 | B |

| Play single named note |
| --- | --- |
| D | D |
| A | A |
| C | C |
| E | E |
| F | F |
| B | B |
| G | G |

## Requirements:

* Python
* NumPY
* sounddevice
