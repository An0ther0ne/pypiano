# pypiano

Play generated sine wave sound with Python and Sounddevice module.
Initialy octave number is 4.
Initialy sound 

## Control keys:

| Key | Desctiption |
| --- | --- |
| ESC | Exit |
| + | Increase volume |
| - | Decrease volume |
| PgUp | Octave + 1 |
| PgDn | Octave - 1 |
| Up | Current frequency + 1% |
| Down | Current frequency - 1% |
